# How to use

Download the newest release zip file from this [link](https://github.com/0xFLOATINGPOINT/Renderite.Proxy/releases)
Extract the `Renderer` folder held within the zip archive to the root of your Resonite installation
Start Resonite as per normal
*profit*?

# How to Build

Clone repo via `git clone` or download via the Github website or use [GitHub Desktop](https://desktop.github.com/download/) (Recommended)
Open `Renderite.Proxy.sln` with Visual Studio 26
In the menu bar navgate to `Build > Publish Selection` Then in the publish page press `Publish` on the top right
Executable will be located in `builds\(TargetRuntime)\Renderer`

# FAQ

What is this app useful for?: It was created to make FrooxEngines Renderer watchdog *go away* without using mods/patchs to game files
What other uses does this app have?: Being able to start any custom renderer with a delayed startup. This includes running Renderite.Renderer directly within the Unity Editor or for developing custom renderers that aim to use the native starting process that Resonite uses for the current Unity Renderer
